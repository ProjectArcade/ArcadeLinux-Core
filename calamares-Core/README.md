# ArcadeLinux-Core

Calamares installer configuration and branding for Arcade Linux.

## Structure
- `calamares/` — settings, branding, and module configs
- `assets/` — logos and wallpapers
- `build-tools/` — helper scripts

## Testing locally
```bash
sudo bash build-tools/install-to-system.sh
sudo calamares -D
```
